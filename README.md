# (M)inecraft (S)erver's (S)tatus (V)iewer ℹ️
 - Shows number and names of players that are online
 - Shows latency (ms)
 - Shows server's version

## Installation guide:
  1. Copy this repo
    `git clone https://github.com/RisetrokX/MSSV.git`
  2. Change directory to this repo
    `cd MSSV`
  3. Install requirements
     `pip install -r requirements.txt`
## Usage
  Run the python script and enjoy MSSV
    `python mc_check.py`
## Why use?
  You can use it if you don't want to open Minecraft and just want to see server's status
